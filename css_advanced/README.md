CSS Advanced - Smile School Website
Project Description
This project involves building a responsive landing page for "Smile School" purely using HTML and CSS, based on a provided Figma design. The primary focus of this project is to apply advanced CSS styling techniques to accurately replicate the design mockups, ensuring a pixel-perfect and responsive layout across different screen sizes.

Learning Objectives
Upon completing this project, I am expected to understand and be able to explain the following concepts without external help:

What is CSS: A styling language used to describe the presentation of a document written in a markup language.

How to add style to an element: Various methods including inline styles, internal stylesheets, and external stylesheets.

What is a class: An attribute used in HTML to define a group of elements that share a common style.

What is a selector: Patterns used to select the HTML elements you want to style (e.g., tag, class, ID, attribute selectors).

How to compute CSS Specificity Value: The algorithm used by browsers to determine which CSS declaration is most relevant to an element and therefore applied.

What are Box properties in CSS: Understanding the CSS Box Model, including content, padding, border, and margin.

How does the browser load a webpage: The process from DNS resolution to rendering the page, including parsing HTML, building the DOM and CSSOM, and creating the render tree.

Requirements
All files end with a new line.

A README.md file at the root of the project folder.

No external libraries, frameworks (like NodeJS, React, VueJS, Bootstrap), or preprocessors are used. The website is built with only HTML, CSS, and JavaScript (though no custom JavaScript logic is required for the styling aspects of this project).

Code should be W3C compliant and validate with W3C-Validator.

Setup and Usage
To view this project locally:

Clone the repository:

git clone https://github.com/ronaldbett64/alx_html_css

Navigate to the project directory:

cd alx_html_css/css_advanced

Open index.html: Locate the index.html file in your file explorer and open it with your preferred web browser.

Ensure that the images folder (containing Path.png, Bitmap.png, smile.svg, 1 (3).png, 6.png, 7.png, 8.png, Bitmap (1).png, Bitmap (4).png, Bitmap (3).png, Bitmap (2).png, and SmileSchoolLogo.png) is present in the same directory as index.html for all images to display correctly.

Design Reference
The design for this website is based on a Figma file provided during the project. The fonts used in the design are:

Source Sans Pro

Spin Cycle OT (For headings, Red Hat Display is used as a similar Google Fonts alternative in the HTML due to Spin Cycle OT not being readily available via Google Fonts. For perfect replication, Spin Cycle OT would require custom font hosting.)
